## Implementation context

- common.mk holds the wx-free common base (COVERAGE block, .DEFAULT_GOAL, base
  CXXFLAGS with -std=c++17, DEPFLAGS/PICFLAGS, suffix rules, shared
  objs/coverage-clean/clean targets). It defines WX_CXXFLAGS LAZILY (recursive
  `=`) as `$(shell wx-config --cxxflags) $(shell pkg-config --cflags gtk+-3.0)`.
  The base CXXFLAGS does NOT reference WX_CXXFLAGS, so pure modules never trigger
  wx-config. Only src/tactical appends `CXXFLAGS += $(WX_CXXFLAGS)`.
- The six standalone Makefiles (src/, src/gui, src/battleSim, tests/, tests/gui,
  tests/tactical) keep their own form and just gained inline `-std=c++17`.
- Preserved: SKIP_MODEL_DEPS, the `.RECIPEPREFIX = >` style in tests/gui and
  tests/tactical, the coverage opt-in, all target names, and every
  -D*_TEST_REPO_ROOT define.
- Edge case to watch: the five pure modules must invoke wx-config ZERO times.
  Confirm with a dry run, e.g. delete one object and run
  `make -C src/core -n FPoint.o` — the compile line must contain no wx-config.
