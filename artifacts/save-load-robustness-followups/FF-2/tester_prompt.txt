  for every scalar read and `if (w->load(is)) { delete w; return 1; }` /
  `if (d->load(is)) { delete d; return 1; }` for the nested weapon/defense loads.

  NULL (never dangling). No leak/double-free.

  abort are UNCHANGED.

FVehicle SERIALIZED FIELD ORDER (src/ships/FVehicle.cpp) — needed to truncate
strictly inside a vehicle's own record. Note the ship TYPE string is written by

itself begins at m_ID. Order written by save():

    write(m_currentHP)         -> sizeof(unsigned int)
    write(m_speed)             -> sizeof(unsigned int)
    write(m_heading)           -> sizeof(int)
    writeU32(m_owner)          -> 4 bytes
    write(m_currentDCR)        -> ...
    write(m_maskingScreenTurnCount)
    write(m_combatControlDamaged) (bool)
    write(m_onFire) (bool)
    write(m_powerSystemDamaged) (bool)
    write(m_navError)
    writeU32(weaponCount)      -> 4 bytes
