Files changed by the Implementer (inspect for the fix):
- src/core/FGameConfig.cpp
    - constructor captures readlink() return in a signed ssize_t, requests

      computeSafeTerminatorIndex(), and on failure (size <= 0) clears
      m_basePath / m_executablePath.

      size_t bufferCapacity): returns 0 for bufferCapacity==0 or
      readlinkResult<=0, otherwise clamps to bufferCapacity-1 when the raw
      result is at/above capacity.
- include/core/FGameConfig.h
    - public static declaration + Doxygen for computeSafeTerminatorIndex();
      updated constructor Doxygen.
